Acerca de PyLaPaz

¿Qué es Verificador de Serie?

Verificador de Serie es una aplicación web desarrollada por la comunidad Python La Paz, orientada a apoyar la verificación referencial de series de billetes "B" en circulación, de acuerdo con comunicados públicos. La aplicación no es un producto oficial de ninguna entidad financiera ni reemplaza los canales formales de verificación.

¿Quiénes somos?

Python La Paz es una comunidad abierta de desarrolladores de Python que promueve el aprendizaje, la colaboración y la creación de herramientas tecnológicas con impacto local.

Características principales

  • Captura con cámara: Uso directo de la cámara del dispositivo
  • Selección desde galería: Carga de imágenes existentes
  • Detección de serie: Lectura asistida por OCR

Tecnología

  • Backend: FastAPI (Python)
  • Frontend: HTML5, CSS3, JavaScript (vanilla)

Privacidad y seguridad

El proyecto adopta un enfoque de mínima recolección de datos. La información técnica se utiliza únicamente para fines operativos y estadísticos internos. No se comparten datos con terceros.

  • Conexiones cifradas en tránsito
  • Código abierto para auditoría comunitaria

Proyecto de código abierto

El desarrollo es comunitario y abierto. Puedes revisar el código, proponer mejoras o reportar problemas en GitHub.
