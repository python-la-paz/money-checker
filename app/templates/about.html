Acerca del Proyecto

1. ¿Quiénes somos?

Python La Paz es una comunidad abierta de desarrolladores que promueve el aprendizaje, la colaboración y la creación de herramientas tecnológicas con impacto local. Este proyecto nace como una iniciativa comunitaria en respuesta a una necesidad ciudadana concreta.

2. ¿Qué es Billete Verificador?

Billete Verificador es una aplicación web desarrollada por Python La Paz, diseñada para apoyar la verificación referencial de series de billetes en circulación, tomando como base los comunicados públicos emitidos por el Banco Central de Bolivia (BCB).

Es importante destacar que esta aplicación no es un producto oficial de ninguna entidad financiera ni reemplaza los canales formales de verificación.

3. Características Principales

  • Captura con cámara: Permite fotografiar el billete directamente desde el dispositivo
  • Selección desde galería: Admite la carga de imágenes existentes para su análisis
  • Detección de serie: Identifica el número de serie mediante lectura asistida por OCR
  • Verificación referencial: Contrasta la serie detectada con las listas de comunicados oficiales

4. Tecnología Utilizada

  • Backend: FastAPI (Python)
  • Frontend: HTML5, CSS3, JavaScript (vanilla)
  • OCR: Procesamiento de imágenes para extracción de texto

5. Código Abierto

El desarrollo de Billete Verificador es comunitario y completamente abierto. El código fuente está disponible en GitHub, donde cualquier persona puede revisarlo, proponer mejoras o reportar problemas. La transparencia del código permite además la auditoría comunitaria del proyecto.

6. Privacidad y Seguridad

El proyecto adopta un enfoque de mínima recolección de datos. La información técnica registrada se utiliza únicamente con fines operativos y estadísticos internos; no se comparte con terceros bajo ninguna circunstancia.

  • Conexiones cifradas en tránsito
  • Código abierto para auditoría comunitaria
  • Sin recolección de datos personales identificables

Para más detalles, consulte nuestro Aviso Legal y de Privacidad.
